## 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/Sulekha98singh/my-portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
